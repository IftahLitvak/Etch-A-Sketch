Welcome to Etch A Sketch! This is a drawing game with a variety of drawing functions, down below at the bottom of the page is a short description about each button, if you are having issues understanding what each button does you are welcome to read it.

  • Hold Mouse Clicked - by default the drawing method is set to 'Hold Mouse Clicked' which means you need to press the mouse and hold for drawing, release the mouse button to stop. When th drawing method is set to 'Click Draw and Click' it means you need to press the mouse button (and release it), then draw and press the mouse button again to stop.
  • Multi Colored Mode - Pressing this will make your drawing be colorfull.
  • Shading Mode - Pressing this will make you shade each grid square you draw.
  • Lighten Mode - Pressing this will make you lighten each grid square you draw.
  • Eraser - Pressing this will make you able to erase grid squars that you draw (they will be colored in the background color).
  • Clear Grid - Pressing this will remove everything you draw.
  • Show Grid Lines - Pressing this will make the grid square borders show up.
  • Size Slider - You can change the size of the grid by moving the slider.
  • Color Mode - Pressing this will make you draw reguraly with a color to your choice.
  • Background Color & Pen Color - Changing the background color will change the color of your grid background and changing the pen color will make you able to draw in 'Color Mode' with your chosen color.
